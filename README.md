# NovaShift
NovaShift is a modern website built with Next.js, showcasing AI automation services that empower businesses to build smarter and move faster. It highlights AI strategy consulting, custom development, workflow automations, and integrations to drive growth, efficiency, and innovation.


## Quick Links
- [Live Demo](https://novashift.vercel.app/)  - Deployed version on Vercel.
- [GitHub Repository](https://github.com/llreebor/novashift) - Source code, issues, and contributions.


## Getting Started
### Prerequisites
* Node.js (version 18 or later)

### Prerequisites
#### Installation
1. Clone the repository:

```bash 
git clone https://github.com/llreebor/novashift.git

cd novashift 
```
2. Install dependencies:

```bash 
npm install
# or
yarn install
# or
pnpm install
```

#### Running the Development Server
Start the development server:
```bash 
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
